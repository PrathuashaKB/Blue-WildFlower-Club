# Blue-WildFlower-Club
Blue-WildFlower-Club, a website that contains a collection of blue flowers.

### About :
This project is about the website containing the collection of information of all the blue wildflowers. The users of this website could be able to access or view the content both for free as well as attaining a vaild subscription by being the member of Blue-WildFlower-Club.  

<span style="color:blue">**This Website is built for following purpose:**</span>

 1. Sign-Up and Login to access the features of website.

 2. Get subscription to access the pro-features.

 3. View information about the vast collection of blue flowers.

 4. Contact and know about us.

### Technologies used :

 1. HTML

 2. CSS

 3. JavaScript

 4. Bootstrap

### Software required :

 1. Visual Source Code(Editor)

 2. Web Browser
